# ChatGPT
A browser-based AI chatbot built using WebLLM, allowing users to run a large language model directly in the web browser â€” no server required. This project demonstrates how to integrate WebLLM into a simple chat interface to provide interactive, ChatGPT-style conversations without sending data to external servers.

# Features

Runs fully in-browser with WebLLM (no API keys required)

Interactive chat interface with conversation history

Responsive layout suitable for both desktop and mobile

Privacy-focused: all inference runs locally on the client device

Tech Stack

React + Vite for frontend development

WebLLM for on-device model execution

CSS Flexbox for layout and styling

# Getting Started
Clone the repository
git clone https://github.com/your-username/chatgpt-webllm.git
cd chatgpt-webllm

Install dependencies
npm install

Run the development server
npm run dev


The application will be available at http://localhost:5173.
